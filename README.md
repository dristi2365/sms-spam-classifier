\# SMS Spam Classifier



A beginner-friendly machine learning project that detects whether a text message is \*\*Spam\*\* or \*\*Ham (not spam)\*\*.



\## Features

\- Trains quickly using a real SMS dataset

\- TF-IDF vectorization + Naive Bayes classifier

\- Interactive message testing

\- Accuracy: ~97–99%

\- Perfect for beginners and GitHub portfolios
